1 Introduction

This document builds the Topoligical Sort Exercise interactive artefact(HTML)

2 Features of the artefact

  • Artefact provides a practice module for Topoligical Sort Algorithm
  • User can see a randomly generate graph or work on same graph as many times as he need.
  • User can click the New Graph button to work on a new graph at any time.
  • User can click the Reset button to work on same graph at any time.

3.2 Body Section Elements

3.2.1 Instruction box

This is an instruction box, in which the instructions needed for the user to perform the experiments are displayed in a form of list. 

  <div class="instruction-box"id="instructions">
    <button class="collapsible"> Instructions </button>
    <div class ="content">
      <ul>
        <li>Select the node that can be added to topologically sorted order.</li>
        <li>Click on <b>Reset</b> to start over with a same graph</li>
        <li>Click on <b>New Graph</b> to start over with a new graph</li>
        <li>Zoom or Adjust graph by dragging nodes according to your convience</li>  
      </ul> 
    </div>
  </div>

3.2.2 Question

This is question user should answer. 

  <div id="que">
    <b>Question :</b> An arrow from 'a' to 'b' represents that
    'b' can be done only after finishing 'a'.  The given
    graphs indicate different jobs of a company. Help the
    company manager assign jobs in an order so that no
    employee needs to wait for other job to be done.
  </div>
